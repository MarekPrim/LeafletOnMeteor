LeafletOnMeteor is an implementation of Leaftlet on Meteor.js isomorphic framework.


## Tech

This repository uses a number of open source projects to work properly:

- [ReactJS] - HTML enhanced for web apps!
- [Twitter Bootstrap] - great UI boilerplate for modern web apps
- [node.js] - evented I/O for the backend
- [Express] - fast node.js network app framework [@tjholowaychuk]
- [MeteorJS] - An isomorphic JS framework


## Installation

This requires [Node.js](https://nodejs.org/) v10+ to run.

Install the dependencies and devDependencies and start the server.

```sh
cd app
npm i
metor run
```
